What are the differences between relative, absolute, and fixed positioning in CSS?

Fri 20 Jan 2023 | Sprint 2

Below I've attempted to explain three of the main positioning options for elements using CSS styling.

These green boxes are parent elements - they each have a number of child elements nested within them. The white boxes have no position stated, whereas each of the coloured boxes have a different position set in the CSS style sheet:

absolute, relative or fixed.

Each coloured box would normally sit directly under the white boxes if no position was stated.

You can adjust your browser window or tilt your device to see how each different position affects the location of each box in relation to its parent.

Position: Absolute

The purple box has its position set to absolute. Which means it is positioned based on the location of its parent.

The styling states that its position should be 50px from the left and 0px from the bottom of its parent.

Position: Relative

The orange box has its position set to relative which means it is positioned relative to where it would normally be within its parent.

The styling states that the position of the orange box should have its right side 20px from where it would typically sit.

Position: Fixed

The blue box has its position set to fixed. Which means it is fixed in relation to the screen you are viewing on.

The styling states that the position of the blue box should be 50px from the bottom and 20px from the right of the screen.
